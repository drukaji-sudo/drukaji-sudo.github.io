# Frost Tower Live

Version corrigée complète.

## À faire

1. Upload tous les fichiers sur GitHub Pages.
2. Dans Supabase > SQL Editor, exécute `supabase-db-live.sql`.
3. Ouvre deux navigateurs/fenêtres avec la même room.
4. Clique Jouer.

## Inclus

- Jeu démarre correctement.
- Table Supabase DB pour le live.
- Liste des joueurs dans la room.
- Classement live.
- Positions synchronisées.
- Même map/platformes pour tous dans la même room.
- Personnalisation : couleur, yeux, style.
